# PyTorch implementation of CNN VAE making use of PyTorch lightning 
```
Pytorch implementation of a convolutional variational autoencoder. PyTorch lightning is used and is a framwork that allows for rapid experimnetation without having to worry too much about the engineering side of the project 
```
## Install the requirements in a virtual environment of choice

```
pip install -r requirements.txt
```

## Run the code 

```
python train.py
```

## Use tensorboard to view the results 

```
tensorboard --logdir logbook_cnnvae 
```
